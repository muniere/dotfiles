# dotfiles

## Get started

```bash
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/muniere/dotfiles/master/install.sh)"
```

## Tasks

Each task has following options:

- `-n`: dry-run
- `-v`: verbose

```bash
# link 
deno task link

# unlink
deno task unlink

# status
deno task list
```
