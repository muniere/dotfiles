# Muniere VS Code Theme

A customized Monokai color theme for Visual Studio Code.

## Installation

1. Build and install the theme:
```bash
cd vault/Code/muniere-vscode-theme
make install
```

2. Open Command Palette (Cmd+Shift+P / Ctrl+Shift+P)

3. Type "Color Theme" and select "Preferences: Color Theme"

4. Select "Monokai Muniere"

## Manual Build

If you only want to build the VSIX package without installing:
```bash
make build
```

## Clean

To remove generated VSIX files:
```bash
make clean
```
